## Archivos recomendados

- `profile-main.jpg` - Foto principal de perfil (400x400px)
- `profile-hero.jpg` - Foto para sección hero (600x600px)
- `profile-about.jpg` - Foto para sección about (300x300px)

## Uso

```html
<img src="./images/profile/profile-main.jpg" alt="Alberto Marturelo Lorenzo" style="width: 150px; height: 150px; border-radius: 50%;">
```

## Recomendaciones

- **Formato**: JPG o PNG
- **Tamaño**: Cuadrado (1:1 aspect ratio)
- **Calidad**: Alta resolución para web
- **Fondo**: Profesional, preferiblemente neutro
- **Expresión**: Profesional y amigable
